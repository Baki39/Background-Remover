# AI Image Enhancer

A production-ready Android app built with Kotlin + Jetpack Compose.

## Features

- ✨ AI-powered photo enhancement
- 🔍 Remove blur & noise
- 📈 Upscale resolution
- 📊 Before/After comparison
- 📱 AdMob integration (Banner + Interstitial)
- 🔐 Firebase Analytics

## Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 34 (Android 14)
- **Architecture:** MVVM + Clean Architecture

## Setup

1. Install Android Studio
2. Clone this repository
3. Add your `google-services.json` (from Firebase Console)
4. Update AdMob App ID in `AndroidManifest.xml`
5. Build and run

## Build

```bash
./gradlew assembleDebug
```

## APK Output

The debug APK will be at: `app/build/outputs/apk/debug/app-debug.apk`

## Store Assets

- App Icon: 512x512 PNG
- Feature Graphic: 1024x500 PNG
- Screenshots: 8 screenshots (1080x1920)
